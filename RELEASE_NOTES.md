# Release Notes — v1.0.0

Release: v1.0.0 — initial lab snapshot
Date: 2025-12-19

Summary

This release contains the collected Packet Tracer lab files for the CAP478: Data Communication and Networking Laboratory course at Lovely Professional University. It includes topology examples, routing labs (static and RIP), subnetting exercises, and application service configurations (HTTP, DNS, DHCP, SMTP, FTP).

Contents (high level)

- Topologies: hub, star, mesh, hybrid examples
- Routing: single-router, multi-router, static routing, dynamic routing (RIP)
- Subnetting: FLSM exercises and solved examples
- Services: HTTP, DNS, DHCP, FTP, Email (SMTP)

Notes for instructors/students

- Open `.pkt` files using Cisco Packet Tracer (v8.x recommended).
- Use Simulation mode to inspect PDUs and observe routing updates.
- Verify services using the verification steps in `LAB_INDEX.md`.
